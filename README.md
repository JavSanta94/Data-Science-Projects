# Data-Science-Projects

Repositorio que agrupa **proyectos integrales de ciencia de datos** y **pruebas técnicas** realizadas en procesos de selección. Cada notebook resuelve un problema real o planteado en entrevistas, mostrando el flujo completo: desde el análisis exploratorio hasta la entrega de insights y modelo final.

> "Este repositorio es mi vitrina práctica en Machine Learning. Contiene tanto proyectos personales donde aplico metodologías end-to-end, como pruebas técnicas reales de procesos de selección. El objetivo es demostrar mi capacidad para entender problemas de negocio, transformar datos, construir modelos robustos y comunicar resultados de forma clara."

## 📂 Estructura
Data-Science-Projects/
├── proyectos/ # Proyectos personales completos
├── pruebas-tecnicas/ # Challenges y evaluaciones de postulaciones
├── datasets/ # Conjuntos de datos (pequeños o de muestra)
└── utils/ # Funciones auxiliares reutilizables

## 🧠 Temáticas cubiertas

- **Regresión** (lineal, árboles, regularización)
- **Clasificación** (logística, random forest, XGBoost)
- **Clustering** (K-means, DBSCAN)
- **Feature engineering** y selección de variables
- **Evaluación de modelos** (métricas, validación cruzada, overfitting)
- **Interpretabilidad** (SHAP, feature importance)
- **ML Pipelines** (scikit-learn pipelines, ColumnTransformer)

## 🛠️ Tecnologías

- Python 3.8+
- pandas, numpy, scikit-learn, xgboost, lightgbm
- matplotlib, seaborn, plotly
- Jupyter Notebook / JupyterLab
